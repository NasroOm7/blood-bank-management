Mise en service du logiciel
Avant de commencer il faut installer le GIT  via le site https://git-scm.com/  en suite  le node js  via le site https://nodejs.org/fr/.
Déplacez-vous vers le dossier du node js dans Terminal installé Électron avec la commande suivante :
--------------
  npm install g electron                                                                                                     
-------------
 Exécutez ensuite les commandes suivantes afin de lancer le projet :

-----------
git clone https://github.com/electron/electron-quick-start
-------------------------
cd electron-quick-start
-------------------------
npm install && npm start 

-------Concernant le code source-------
Vous trouvez dans le css-js.zip les fichiers css,js,img,bootsrap...
et dans le app.rar vous trouvez le code php et toutes les pages du logiciel.
